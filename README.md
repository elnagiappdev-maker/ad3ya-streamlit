# أدعية صوتية – النسخة النهائية

## كيفية الاستخدام
1. ضع ملفاتك الصوتية داخل مجلد `audio/` (مثلاً: `سورة الفاتحة شيخ الزين.mp3`).
2. يمكن تعديل الأسماء في `assets/dedications.txt` (سطر لكل اسم).
3. يمكنك إضافة صورة خلفية باسم `bg.jpg` داخل مجلد `assets/`.
4. التطبيق يقرأ فقط — لا يمكن للمستخدمين التعديل.

## التشغيل محلياً
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
